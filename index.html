1 Scan Source Directory

Enter the full path to the folder containing your messy media files. The app will generate a command to safely map the files.

Run this command in PowerShell. It will scan the folder and copy the file list to your clipboard.

Get-ChildItem -Path "C:\Downloads" -File -Recurse | Select-Object FullName, Name, Length | ConvertTo-Json | Set-Clipboard

2 Resolve Metadata

Paste the clipboard contents (the JSON payload) from Step 1 into the box below. The system will automatically begin processing.

Communicating with TMDB API...

3 Define Destination & Execute

Enter the root path of your final media library. The script will automatically create the required Plex-friendly subfolders here.
